    # 动态效果

    默认的数据是固定的，如果需要动态加载数据，请使用websocket(或者socket.io获取更多兼容性)或者轮询(可能会消耗大量的请求资源)，echarts在每次渲染的时候自动化有一个缓动效果的动画。

    长链接需要后端支持

    如果需要假数据，又不想用请求的话，请使用mock数据或者定时器加载，完成简单的动态效果。

    简单说明，很多特效对于电脑性能的要求比较高，用户使用的过程中可以检测一下特效对于内存以及CPU的占用情况，可以在chorme更多工具>任务管理器内查看具体页面的使用情况
