    # Bmap 入门课程

    • 由于世界地图配置的复杂以及不同需求的实现，期望用户可以自行探索地图的可能性

    项目推荐使用Bmap,申请账号之后

    bmap配置课程

    由于目前Bmap相关的资源全网都较少，目前使用频次较低，开发难度也不大，难免有不足之处，烦请自行找解决方案。

    而且目前Bmap也在迭代阶段，还不是很成熟，很多特性还不能支持的很好，对地图要求较大的可以参考一下其他开放API的地图(高德或者腾讯)

    简单的配置可以参考 网上 现有的 教程 视频 目前 缺失较多
