# 数据可视化开箱方案-vue

vue项目的启动需要本地启用一个npm服务进行页面的渲染

  • 初学者

官方文档——vue项目起手 (opens new window)

# 启动项目


项目运行之前，请确保系统已经安装以下应用

1、node (6.0 及以上版本)
npm install 或 yarn(推荐) //安装依赖 

npm run start  //启动项目

npm run build  //如果需要打包 默认在文件夹的/dist 中

# 调试项目

目前vue项目都有热加载的依赖项，所以直接通过修改页面和数据就能看到对应的变化，目前组件库主要是使用了dataV和echarts，链接如下.

datav大屏可视化组件 (opens new window)

echarts官方配置项手册 (opens new window)

如果需要不同的样式可以在echarts示例中修改，然后将原文件的options替换

  • 百度官方Bmap

百度官方的BmapApi (opens new window)

如果需要自定义百度地图请参考API,为了不影响各自使用，请独立申请百度API的key,多人使用可能会导致加载的异常

方法链接:

https://jingyan.baidu.com/article/e3c78d64ffd53e3c4c85f585.html

使用http-server启动本地服务进行调试

npm页面的http-server说明 (opens new window)

因为有些页面资源需要动态加载，只有部署到服务上才能查看，比如Bmap，请本地启用一个node服务进行页面调试，

由于项目配置差异.无法热加载请手动刷新页面。

element-ui (opens new window) 供Vue使用
