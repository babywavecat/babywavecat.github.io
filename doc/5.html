    # 建站产品介绍

    本工作室承接各种企业，个体户官网建设。帮助您推广介绍您的产品，通过搭建万维网的形式，将用户的信息发布到互联网上，可以直观明了的查看关于您企业的信息 以及经营状况，同时也可以发布一些招聘信息以及展示企业风采。

    建设包含移动端，PC端等页面，包含企业的大部分介绍信息，对外公开的信息以供展示。开发工期7-30天。

    如果您的企业具有出海业务，需要特别备注。

    # 收费构成

    目前团队只能开具自然人发票，由税务局代开发票，请知悉。工商个体户还在注册中，如有麻烦请见谅。

    如果仅需要简单的官网进行展示，本团队有成熟的官网解决方案，一次性收费1000元，后续维护按照更新内容另行收费。

    如果不需要云服务器成本进行官网搭建，本团队同样有成熟的官网部署解决方案，一次性收费2000元(顶级域名除外)，涵盖一年的域名申请费用以及维护域名解析等 费用。

    另，如果申请的域名一年使用价值超过800元/年，收费公式(1200+一年域名适用费用*1.2)

    需要服务器搭建以及自定义官网成本等，最低套餐价1w/年，具体包含：

    • 一年的服务器使用成本
    • 一年域名适用
    • 一年20次维护更新费用

    另，如果需要增量服务器成本以及域名，收费公式(8200+一年域名适用费用 * 1.2+一年的服务器使用成本 * 1.5)

    # 域名/服务器介绍

    本团队通过采购主流的云服务器进行网站的搭建，此项成本包含在开发成本里面。可以独立开票。

    目前支持的云平台有阿里云，华为云，腾讯云，百度云等主流的云服务厂商，由于涉及到备案问题，需要您本人或者公司法人配合进行备案相关行为，我方人员

    会同步的进行指导。

    # 具体流程

    1. 协商具体的需求以及费用问题

    2. 出具相关的企业介绍文案，官网展示的物料，照片，logo图标,联系电话，地址，微信公众号等，以及公司对应的资质证明(若需要)

    (如果公司缺少各种宣传介绍等，我方运营人员将会进行指导，教您提供相关的内容解决方案)

    1. 拟一份合同草案，双方签订，付定金

    2. 甲方交付产品，乙方付尾款

    # 百度统计

    部署本项目，将为您自动添加百度统计，方便收录到百度搜索中，提高您的网站排名，同时也会将您的网站访问指标可视化的展示出来。

    如果需要，可以留下对应运营人员的账号/密码，将为您接入百度统计的指标信息，自己登陆后台即可查看。

    # 百度客服

    如果您的公司有业务需要，可以接入百度客户，通过点击客服按钮唤起，一对一沟通。

    此项目为增值项目，具体收费细则由百度公司提供，我方提供日常维护以及对接需求。

    如果需要运营微信公众号以及小程序开发等，请另行咨询。
