# Step seven. 21年10月so. far. so. good. 总结

目前来说已经工作了几乎三年了，还有一个月完整的三年直接经历，全职的工作直接没能让我过上理想的生活，消费水平确实提升了不少，但是信用额度还是没能和个人能力匹配，直接完整的踏步和梦游好久，被干扰好多无用的信息，不知所云。

今年也马上就要过去，专业的一年尝试了好多实践，还是缺点意思。

工作的能力还是存在自己的边界之中，也没能突破，心态也随着听说的事情变得微妙起来，不切实际想要起飞。

21年剩下的2个月，超越能力的臆想工作直接拒绝，好好面对工作和生活。坚持一个合理的计划表，系统的把谦虚和手头的工作再安排解决，完成简单的年度小目标，设置一个2022大年icon，达成之后再去想象可能性。拒绝工作日的聊天以及无聊的视频流，直接学习减肥。整理生活，完成长期的上班待业中的routine日常，直接社会工作研究生。

1.每天完成puzzle或者解题新技术，直接输出文档。

# 性质

外企的特质应该不会激进的扩张，所以核心还会是处理问题的方法和习惯，更多的时间自由思考工作行业和生活走向，对自身的发展也是长期有利。

# schdule

工作日大头基本上还是office time以及两段commuting time，上班的时间focus有限，早前都花在了碎片资讯的阅读上面，精进专业方面还是有所欠缺。

  • commuting time 阅读和解系列，直接与自己和解

  • office time 完成工作之余持续建设项目以及个人需求池，减少碎片资讯的阅读量，在项目实践以及系统学习中make change。

  • off hour 不脱离工作日重心，回家找到有趣的持续构建驱动力，完成阶段性的一个sample icon

# sample icon

  • 工作方面 大概的icon即是，本年度结束前系统的补充完职业基础知识，改善学习方法和心态，持续进步。

  • 达成 or 个人能力储备达标 直接达成小而美公司的offer once,在团队中学习中驱动提升，同时也没有互联网公司的内卷，兴趣导向的持续学习。
